JavaScript Basics

JavaScript is a cross-platform, object-orientes scripting languaje. JavaScript is extremely popular for a variety reasons. It is a small and lightweight language allowing maximum flixibility for developers to take in a bunch of diferent directions. JavaScript lives inside a host evironment (a web browser or Node server), it cant be connected to the objects of these evironments to provide programmatic control over them.

  • Variable Declaration JavaScript variables are containers for storing data values - imagine a cup you fill with the coffee, the cup holds the coffee, a variable holds a value. All JavaScript variables must be identifed with unique names. These unique names are called identifiers.var x

  • Variable AssignmentAssiggnament operators assign values to JavaScript variables-our cup can now have coffee poured in it, giving our variable a value to hold. The=assignament operator assign a value to a variable.var x = 10

  • Data TypesData types are an important concept; to be able to operate on variables you need to know the data type. There are six data types that are JavaScript primitives: Boolean-true or false; null- nullaka nothing; Number- 42 or 3.14159; String- "Coding Dojo Rocks!"; Arrat- [1,"Coding",2,"Dojo"]; and Objet- {first_name:"Jane", last_name: "Doe"}

  • ConditionalsWhen you write code, you want to perform different actions for differents decisions-hitting different code blocks based on values or conditions that have been met. You can use conditional statements in your code to accomplish this. There are following conditionals statements: if a specified condition is true, do this code in our code block; else if to specify a new condition to test, if the first condition is false;elsewe execute this block of code;

  • LoopsThere are many different kinds of loops in every programming language, but they all essentially do the same thing: they will repeat an action some numbers of times. Imagine you have to run a mile, well you run around the track four times and then you stop. Thats a loop!

  • FunctionFunction are an en encaspulation of a code block. When we call our function this will run that code block. Think of as a list of intructions.
